## 📖 About Abhira.io

**Abhira.io** is an advanced AI-powered website builder designed to transform ideas into fully functional websites within seconds.
It removes the complexity of coding by allowing users to simply describe their requirements, and the AI generates a complete, responsive website automatically.

The platform is built to empower:

* 🚀 **Startups** to quickly launch their online presence
* 💼 **Businesses** to build professional websites effortlessly
* 👨‍💻 **Developers** to speed up prototyping and development

With a strong focus on **automation, scalability, and user experience**, Abhira.io bridges the gap between creativity and technology.

---

## ⭐ Key Features

### 🤖 AI Generation

Describe your idea and AI generates a complete landing page in seconds.

### 📊 User Dashboard

Manage projects, edit configurations, and monitor performance.

### 📱 Responsive Design

All generated websites are fully responsive across mobile, tablet, and desktop devices.

### 🔐 Secure Authentication

JWT-based authentication ensures secure and reliable data handling.

* React.js

* Node.js

* MongoDB

**Deployment:**

## ⚙️ Local Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/abhishekshivahre08/Abhira.io.git
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run Development Server

```bash
npm run dev
```
